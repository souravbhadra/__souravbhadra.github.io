Projects

Seed Composition Estimation using CNN

  • Formulate a voxelization algorithm to represent LiDAR point cloud as sparse matrix. Utilizing a sparse CNN to estimate canopy architectural traits of corn and sorghum from the sparse matrix.
  • Devised a direct imagery-based prediction model using 2D and 3D variants of ResNet and VGGNet to estimate seed composition and yield of in-field soybean and corn.

PROSAIL-Net: A Transfer Learning DNN for Corn Biophysical Traits

Generate a transfer learning-based PROSAIL inversion technique that estimates different plant phenotypes/traits from push-broom hyperspectral and LiDAR in UAV platforms.
The automated workflow involves machine learning pipelines to predict different plant chemical properties using support vector machine, random forest and partial least squares regression.
Solar-sensor zenith and azimuth angles are calcualted using open source Python packages (PVLIB, Rasterio, Geopandas), linear algebra (SciPy) and trigonometric functions.

TERRA-REF

The project transformed plant breeding by incorporating big data analytics, machine learning, and remote sensing. A large mechanical crane (Gantry Scanalyzer) with a handful number of sensors (hyperspectral, RGB, and thermal) continuously scans crops underneath it.
Developed a soil and shadow removal algorithm using physical rules of solar radiation and machine learning for the automated pipeline.
Trained a deep neural network to predict very near and short-wave infrared irradiance and reflectance.

Hydrologic Connectivity using AI

The goal of this NSF funded project was to improve the hydrologic connectivity from high-resolution digital elevation models (DEMs) using Convolutional Neural Networks (CNN) and Deep Neural Networks.
